从零开始学习大模型
第一天 ： 学习线性模型 
1.需要建立训练集，存在过拟合的状态，所以会有训练集和开发集
2.评估误差建立loss,MSE = （Σ所有误差的平方）/ 测试的数量，越接近0拟合效果越好
3.利用python建立模型，然后利用matplotlib的2D或者3D的绘图包进行绘画，因变量是MSE 
4.利用numpy的unique函数可以对某个集合化简至唯一值并进行排序，比如{3，2，2}化为{2，3}
5.利用numpy中的meshgrid,可将两个数组A和B建立一一对应的二元对，若是meshgrid(A,B),则A被扩展成len(B)个相同行的矩阵，B被扩展成len(A)个相同列的矩阵
例如 A=[1,2,3],B=[4,5],A会变成[[1,2,3],[1,2,3]],B会变成[[4,4,4],[5,5,5]] (此时B对应的是行优先，A对应的是列，比如第一行对应（1,4）,（2,4）,(3,4))
